This is a repository containing all the CC-BY licensed datasets from the Semantic Finlex data service at http://data.finlex.fi.

This repository contains data from the following datasets:
Original statutes (CC-BY)
Judgments of the Supreme Court (CC-BY)
Judgments of the Supreme Administrative Court (CC-BY)

Semantic Finlex is developed by the Ministry of Justice in Finland, Aalto University, and Edita Publishing Oy.
